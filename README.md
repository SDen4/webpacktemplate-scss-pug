##  Start new project using this build:
```sh
$ git clone https://github.com/SDen4/webpacktemplate-scss-pug.git
$ cd webpacktemplate-scss-pug
$ npm i
$ git init
$ git remote set-url origin <link of new repo in github>
$ git push -u origin master
$ npm run dev
```
